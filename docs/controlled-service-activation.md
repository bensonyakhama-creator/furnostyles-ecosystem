# Controlled Service Activation

## Repair Requests

### Request Control
- **Submission Control**
  - Only verified customers can submit
  - Detailed problem description required
  - Photo upload required
  - Location details required
  - Urgency level required
  - Budget range required

- **Request Processing**
  - Category classification (automated)
  - Priority assignment (automated)
  - Geographic routing (automated)
  - Provider matching (controlled)
  - Assignment confirmation (manual review for first 3 requests)

### Quality Control
- **Provider Selection**
  - Only verified providers
  - Trust score > 70
  - Performance score > 70
  - Geographic match
  - Skill match
  - Availability check

- **Quality Gates**
  - Pre-service quality check
  - During-service quality monitoring
  - Post-service quality verification
  - Customer satisfaction required
  - Quality score tracking
