# Furnostyles Content Style Guide

**Document Type:** Content Guidelines | **Date:** 2026-05-22 | **Version:** 1.0

---

## How to Use This Document

This document defines the content style guide for Furnostyles. Use it to:

- Maintain consistent tone and voice across all content
- Write premium, luxury content that matches the brand
- Optimize content for SEO while maintaining quality
- Create effective CTAs
- Write engaging blog articles and service descriptions

**Follow this guide for all content creation.**

---

## Furnostyles Tone

### Brand Voice Characteristics

| Characteristic | Description | Example |
|----------------|-------------|---------|
| **Luxurious** | Sophisticated, elegant, premium | "Experience the epitome of luxury living" |
| **Professional** | Knowledgeable, expert, reliable | "Our expert designers bring decades of experience" |
| **Warm** | Welcoming, approachable, friendly | "Let us transform your space into a home" |
| **Confident** | Assured, authoritative, bold | "Kenya's premier interior design destination" |
| **Inspiring** | Motivating, aspirational, visionary | "Discover the art of refined living" |

### Tone Guidelines

| Do | Don't |
|-----|-------|
| Use sophisticated language | Use slang or overly casual language |
| Be professional and expert | Be vague or uncertain |
| Be warm and welcoming | Be cold or distant |
| Be confident and authoritative | Be hesitant or unsure |
| Be inspiring and aspirational | Be mundane or boring |

### Tone by Content Type

| Content Type | Tone |
|--------------|-------|
| **Homepage** | Inspiring, confident, luxurious |
| **Service Pages** | Professional, expert, warm |
| **Blog Articles** | Educational, inspiring, authoritative |
| **Portfolio** | Showcasing, proud, professional |
| **CTAs** | Direct, confident, inviting |

## Luxury/Premium Voice

### Luxury Writing Principles

| Principle | Application |
|-----------|-------------|
| **Elevated Vocabulary** | Use sophisticated words (e.g., "exquisite" instead of "nice") |
| **Sensory Language** | Appeal to senses (e.g., "velvet textures," "warm lighting") |
| **Emotional Connection** | Connect to aspirations and dreams |
| **Exclusivity** | Emphasize uniqueness and rarity |
| **Quality Focus** | Highlight craftsmanship and attention to detail |

### Luxury Vocabulary

| Common Word | Luxury Alternative |
|-------------|-------------------|
| Nice | Exquisite, stunning, beautiful |
| Good | Exceptional, outstanding, superb |
| Big | Spacious, grand, expansive |
| Small | Intimate, cozy, refined |
| Expensive | Premium, investment-worthy, exclusive |
| Cheap | Affordable, accessible, value-driven |

### Luxury Sentence Structure

| Technique | Example |
|------------|---------|
| **Parallelism** | "Design with precision. Live with elegance." |
| **Alliteration** | "Sophisticated spaces, stunning styles." |
| **Rhythm** | "From concept to completion, we craft every detail." |
| **Imagery** | "Golden light floods through floor-to-ceiling windows." |

## SEO Writing Standards

### SEO Writing Principles

| Principle | Application |
|-----------|-------------|
| **Keyword Integration** | Natural keyword placement, no stuffing |
| **User Intent** | Write for user first, SEO second |
| **Structure** | Clear headings, short paragraphs, bullet points |
| **Quality** | Original, valuable, comprehensive content |
| **Readability** | Flesch Reading Ease >= 60 |

### Keyword Guidelines

| Guideline | Implementation |
|-----------|----------------|
| **Primary Keyword** | Include in title, H1, first paragraph, URL |
| **Secondary Keywords** | Include naturally throughout content |
| **Keyword Density** | 1-2% (natural, not forced) |
| **LSI Keywords** | Include related terms and synonyms |
| **Long-Tail Keywords** | Target specific, less competitive terms |

### Content Structure for SEO

| Element | Requirement |
|---------|-------------|
| **Title** | 50-60 characters, includes primary keyword |
| **Meta Description** | 150-160 characters, includes primary keyword |
| **H1** | Includes primary keyword, unique per page |
| **H2s** | Include secondary keywords, logical hierarchy |
| **Internal Links** | 3-5 internal links to related content |
| **External Links** | 2-3 links to authoritative sources |
| **Images** | Optimized WebP, descriptive alt text |

### SEO Content Length

| Content Type | Minimum Length |
|--------------|----------------|
| **Cornerstone Article** | 2000+ words |
| **Standard Article** | 800+ words |
| **Service Page** | 500+ words |
| **Location Page** | 500+ words |
| **Product Description** | 300+ words |

## CTA Tone

### CTA Principles

| Principle | Application |
|-----------|-------------|
| **Action-Oriented** | Use strong verbs (e.g., "Transform," "Discover") |
| **Benefit-Focused** | Highlight value, not just action |
| **Urgency** | Create subtle urgency without pressure |
| **Clarity** | Clear, concise, unambiguous |
| **Confidence** | Assure user of positive outcome |

### CTA Examples

| Context | CTA |
|---------|-----|
| **Homepage Hero** | "Transform Your Space Today" |
| **Service Page** | "Schedule Your Consultation" |
| **Portfolio** | "View Our Latest Projects" |
| **Blog Article** | "Discover More Design Tips" |
| **Contact Page** | "Start Your Design Journey" |

### CTA Button Text

| Weak CTA | Strong CTA |
|-----------|------------|
| Submit | Transform Your Space |
| Click Here | Discover Luxury Living |
| Learn More | Explore Our Services |
| Contact Us | Start Your Project |
| Sign Up | Join Our Community |

## Blog/Article Style

### Article Structure

| Section | Description |
|---------|-------------|
| **Title** | Compelling, includes primary keyword |
| **Introduction** | Hook reader, state value proposition |
| **Body** | Main content, organized with H2s |
| **Conclusion** | Summarize, provide CTA |
| **Author Bio** | Brief author information |
| **Related Articles** | Links to related content |

### Article Writing Guidelines

| Guideline | Implementation |
|-----------|----------------|
| **Hook** | Start with compelling question, statistic, or story |
| **Scannability** | Use short paragraphs, bullet points, subheadings |
| **Examples** | Include real examples and case studies |
| **Quotes** | Include expert quotes for authority |
| **CTA** | End with clear CTA related to article topic |

### Article Tone

| Section | Tone |
|---------|-------|
| **Introduction** | Engaging, curious, promising value |
| **Body** | Educational, authoritative, practical |
| **Conclusion** | Inspiring, actionable, confident |
| **CTA** | Direct, inviting, benefit-focused |

### Article Formatting

| Element | Format |
|---------|--------|
| **Paragraphs** | 2-3 sentences max |
| **Sentences** | 15-20 words average |
| **Headings** | H2 for main sections, H3 for subsections |
| **Bullet Points** | Use for lists, keep parallel structure |
| **Bold/Italic** | Use sparingly for emphasis |

## Service Page Style

### Service Page Structure

| Section | Description |
|---------|-------------|
| **Hero** | Service name, compelling tagline |
| **Introduction** | What the service is, who it's for |
| **Benefits** | Why choose this service |
| **Process** | How the service works (step-by-step) |
| **Pricing** | Pricing information or "contact for quote" |
| **Testimonials** | Client testimonials |
| **FAQ** | Common questions answered |
| **CTA** | Clear CTA to contact or book |

### Service Page Writing Guidelines

| Guideline | Implementation |
|-----------|----------------|
| **Clarity** | Clearly explain what the service is |
| **Benefits** | Focus on client benefits, not features |
| **Process** | Break down into clear, numbered steps |
| **Trust** | Include testimonials, credentials |
| **CTA** | Clear next step (contact, book, quote) |

### Service Page Tone

| Section | Tone |
|---------|-------|
| **Hero** | Confident, inspiring, luxurious |
| **Introduction** | Professional, informative |
| **Benefits** | Persuasive, benefit-focused |
| **Process** | Clear, reassuring, professional |
| **Pricing** | Transparent, value-focused |
| **Testimonials** | Authentic, social proof |
| **FAQ** | Helpful, informative |
| **CTA** | Direct, inviting, confident |

## Portfolio Description Style

### Portfolio Description Structure

| Section | Description |
|---------|-------------|

| **Solution** | How the problem was solved |

### Portfolio Tone

| Section | Tone |
|---------|-------|

| **Challenge** | Empathetic, understanding |
| **Solution** | Expert, confident, creative |
| **Result** | Proud, satisfied, successful |
| **Testimonial** | Authentic, appreciative |

## Email Content Style

### Email Principles

| Principle | Application |
|-----------|-------------|
| **Subject Line** | Compelling, clear, benefit-focused |
| **Personalization** | Use recipient name when possible |
| **Scannability** | Short paragraphs, bullet points |
| **Single CTA** | One clear call-to-action |
| **Mobile-Optimized** | Short subject lines, concise content |

### Email Tone

| Email Type | Tone |
|------------|-------|
| **Welcome** | Warm, welcoming, informative |
| **Newsletter** | Valuable, educational, inspiring |
| **Promotional** | Exciting, benefit-focused, urgent |
| **Transactional** | Clear, professional, informative |

### Email Structure

| Section | Description |
|---------|-------------|
| **Subject Line** | 30-50 characters, compelling |
| **Preheader** | Preview text, reinforces subject |
| **Salutation** | Personalized greeting |
| **Body** | Main content, scannable |
| **CTA** | Clear, single action |
| **Sign-off** | Professional closing |
| **Footer** | Contact info, unsubscribe link |

## Content Quality Checklist

Before publishing any content:

- [ ] Tone matches Furnostyles brand voice
- [ ] Language is sophisticated and professional
- [ ] Content is original and valuable
- [ ] SEO keywords integrated naturally
- [ ] Structure is clear and scannable
- [ ] Headings are logical and descriptive
- [ ] Internal links included (3-5)
- [ ] External links included (2-3)
- [ ] Images optimized (WebP, alt text)
- [ ] CTA is clear and benefit-focused
- [ ] Proofread for grammar and spelling
- [ ] Fact-checked for accuracy
- [ ] Length meets minimum requirements
- [ ] Meta tags optimized
- [ ] Mobile-friendly formatting
